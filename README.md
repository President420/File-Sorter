File Sorter (SorterGUI)

What this is:

This is a simple to use file/folder organiser
there is currently 3 Ways to use source folder

auto/AUTO

All auto does is automatically points to the current users Downloads Folder C:\Users\YOURNAME\Downloads

desktop/Desktop

This does the same as auto but for the desktop

Manually specify Directory

you can use the browse button or paste the Path in the source_Folder field

IMPORTANT:
Basic Tab

Source Folder Selection,
Dry run. 
Enabled By Default(Disable To Actually Use The Sorter)
This option is PREVIEW-ONLY meaning all it does is make a log file in the source folder 
in said log file you can see what files would have been moved/skipped/ignored and where they would have been moved to!

Move Unlisted Folders, looks for directories and gets the size if this is enabled it will move folders/directories aswell 
(this Obeys Size Limit, Disable Size Limit or Change the limit to move large folders Otherwise maximum Folder/Director size that will be moved is 4GB)
